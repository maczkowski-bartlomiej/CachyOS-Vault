# GTK Bookmarks

## Files

```text
Repo source: custom-configs/GTK/bookmarks
Install targets:
  ~/.config/gtk-3.0/bookmarks
  ~/.config/gtk-4.0/bookmarks
```

## Behavior

The installer adds labeled Places entries for:

```text
/mnt/dev
/mnt/dev-data
/mnt/data-hdd
```

Existing unrelated bookmarks are preserved. Existing entries with the same URI are replaced by the managed label.
